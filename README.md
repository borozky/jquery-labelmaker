# LabelMaker

Written by Joshua Orozco (borozky@gmail.com)

## Demo
[https://labelmaker.borozky.com](https://labelmaker.borozky.com)

## Uses
- create-react-app (React is removed)
- ES6
- jQuery
- Redux
- FileSaver.js
- JsBarcode

## Setup locally
```
yarn install
yarn start
```

## Deployment
```
yarn install --production=true
yarn build
```
